# vue-task

## Project setup 2 ways :

## * 1
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## * 2 

### runs with docker file

```
docker-compose up -d --build
```

#### note :
##### the project will run on port: 3000
